Demo Video Script — Neon Tic Tac Toe (3–5 minutes)

0:00 - 0:15 — Title and Objective
- Title slide / quick voice intro: "Neon Tic Tac Toe — a responsive web implementation demonstrating game logic, AI, persistence, and accessibility." 

0:15 - 0:40 — Launch and Start Screen
- Show `tictactoe.html` loaded in browser.
- Point out mode cards (Player vs Player, Player vs AI) and AI difficulty selector.

0:40 - 1:40 — Playthrough (AI Demonstration)
- Choose Player vs AI → Medium/Hard.
- Start a round and demonstrate AI moves, showing differences in behavior.
- Show a win and highlight the winning cells.

1:40 - 2:00 — Stats and Persistence
- Show scoreboard, statistics section, and how data persists after reload.
- Toggle sound off/on to demonstrate the `Sound` control.

2:00 - 2:30 — Export & Reset
- Click `Export Stats` to download JSON and open it briefly to show contents.
- Use `Reset Stats` to clear stored stats (confirm dialog).

2:30 - 3:00 — Accessibility & Mobile
- Demonstrate keyboard navigation (arrow keys + Enter).
- Resize to a mobile viewport and show responsive layout.

3:00 - 3:30 — Code Walkthrough (brief)
- Open `tictactoe.js` and point out: game state, `makeMove()`, `checkWinner()`, AI (`minimax()`), and `recordGameResult()`.
- Mention `localStorage` usage and `exportStats()` implementation.

3:30 - 3:45 — Wrap Up
- Recap features and technologies used.
- Offer to answer questions.

Notes for recording:
- Keep narration concise and focus on showing the app working.
- Capture clear visuals for each screenshot recommendation.
- If live recording is not possible, prepare a screen-capture with short clips for each section above.
